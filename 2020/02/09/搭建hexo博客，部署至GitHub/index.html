搭建hexo博客，部署至GitHub

第一步 预备工作

hexo是基于Node.js的静态博客框架，再博客搭建完成后我们需要使用git将博客部署至github。

所以我们需要事先安装 git 和 Node.js。
安装完node.js后，在cmd中分别输入node -v和包管理器npm -v来查看node和包管理器的版本，如果正确显示出版本，就代表node.js安装无误。

第二步 下载hexo

理论上，我们可以直接用包管理器npm来直接下载hexo，可是使用npm安装插件是从国外服务器下载的，速度很慢而且很不稳定。所以国内的用户可以使用淘宝npm镜像（cnpm)。

  1. 用npm安装cnpm 安装好cpmn后在cmd输入cnpm -v查看其版本，如果能显示版本说明安装无误。
  2. 用cnpm安装hexo

第三步 将hexo安装至本地并启动

  1. 初始化放置hexo文件的文件夹
  2. 启动博客 进入localhost:4000, 这就是你本地的博客了。hexo自动为你生成了第一篇文件helloworld

第四步 部署至github

  1. 登陆到github,新建一个repository，命名必须 为 （你的用户名）.github.io
  2. 用cnpm在block目录下安装一个git部署插件
  3. 配置文件至github
    打开_config.yml, 找到最后的deploy 标注repo（你的github hexo项目的链接）和branch
    如下： [username]是你GitHub的用户名
    1. 部署至远端 按照要求输入GitHub账号
      现在，我们已经把hexo博客部署到GitHub了。在网站栏上输入[username].github.io 就能看到自己的博客了。
