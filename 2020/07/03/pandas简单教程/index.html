import pandas as pd

pandas有两个核心对象： DataFrame 和Series。

DataFrame

​ pd.DataFrame({'Yes': [50, 21], 'No': [131, 2]})

Yes No
0 50 131
1 21 2

DataFrame不仅可以存放integer。

pd.DataFrame({'Bob': ['I liked it.', 'It was awful.'], 'Sue': ['Pretty good.', 'Bland.']})

DataFrame是一个dictionary-list constructor，他给列分配值，序号从0开始每次+1。

我们也可以给每行取名。 通过Index属性。

Series

Series是一组数值。DataFrame是一个表格，Serirs是一个列表。

pd.Series([1, 2, 3, 4, 5])

同样可以用Index给行命名。

pands 中的indexing

1. iloc

​ index-based selection, 就是根据数值位置选取数据。

​ 选取第一行： reviews.iloc[0]

​ 选取第一列： reviews.iloc[:, 0]

​ 选取第一列的第2,3行： reviews.iloc[1:3, 0] 或者 reviews.iloc[[0, 1, 2], 0]

2.loc

​ label-based selection

​ 第一行中country对应的值： reviews.loc[0, 'country']

`

iloc uses the Python stdlib indexing scheme, where the first element of the range is included and the last one excluded. So 0:10 will select entries 0,...,9. loc, meanwhile, indexes inclusively. So 0:10 will select entries 0,...,10.

set_index()方法可以执行某一列作为索引列。

选取reviews中country是Italy的行： reviews.loc[reviews.country == 'Italy']

选取reviews中country是Italy并且points大于90的行： reviews.loc[(reviews.country == 'Italy') & (reviews.points >= 90)]

选取reviews中country是Italy或者points大于90的行： reviews.loc[(reviews.country == 'Italy') | (reviews.points >= 90)]

isin

选取country是Italy或者France的行： reviews.loc[reviews.country.isin(['Italy', 'France'])]

isnull 和 notnull

选择price不是null的行： reviews.loc[reviews.price.notnull()]

分配值

给一列分配一个值： reviews['critic'] = 'everyone'

给一列分配不通知： reviews['index_backwards'] = range(len(reviews), 0, -1)

value_counts() 给出某一列所有元素和各自出现的次数。

Dtypes

DataFrame的某一列或Series的数据类型(data type)叫dtype。

  • 可以用dtype属性获取某一列的数据类型.

    ​ reviews.price.dtype

    ​ output: dtype('float64')

  • 可以用dtypes获取DataFrame每一列的数据类型。

​ reviews.dtypes

​ output：

float64: 64位浮点数(floating point)

int64: 64位整形数。

由字符串组成的列没有自己的数据类型，他们都被定义为 object类型。

astype()方法把一种数据类型转换成另一种数据类型，当这种转换存在的时候。

reviews.points.astype('float64')

  • points属性列本来是int64，被转换成了float64.

index也有自己的dtype。

​ reviews.index.dtype

​ output: dtype('int64')

Pandas也支持一些外来数据类型，比如categorical data和timeseries data。

Missing data

没有值的entry被定义为NaN，“Not a Number”的简写。 由于技术原因，NaN是float64类型。

pd.isnull()

​ reviews[pd.isnull(reviews.country)] ：获取country属性没有值的行。

fillna()

1. 可以把每一个NaN定义为**Unknown**。
2. 给每一个NaN赋值为后面第一个非NaN的值。 这种策略叫 **backfill strategy**。

改变已经存在值。

replace()

​ reviews.taster_twitter_handle.replace("@kerinokeefe", "@kerino")

​ 把taster_twitter_handle属性中的@kerinokeefe全部变成@kerino。

Renaming

rename()

​ 改变index name或者column name。

​ reviews.rename(columns={'points': 'score'})

​ 把points这一列的名字变成score。

​ 如果要改变index name, 格式 rename(index={from, to})

​ reviews.rename(index={0: 'firstEntry', 1: 'secondEntry'})

reviews.rename_axis("wines", axis='rows').rename_axis("fields", axis='columns')
